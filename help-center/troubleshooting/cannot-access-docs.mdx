## Issue

- Page fails to load or returns a 403 error
- You see "Access Denied" message

## Common Causes

- Not signed in
- Missing permissions for the project

## Quick Fix

<Steps>
  <Step title="Log In">
    Ensure you are signed in at `https://serhiilukashov.com/login`.
  </Step>
  <Step title="Request Access">
    Ask the project owner to grant you documentation access.
  </Step>
</Steps>
